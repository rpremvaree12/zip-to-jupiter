ZipGrade -> JupiterGrades

Easily Convert ZipGrade Data for Easy Grade Import to Jupiter!
  1. Download ZipGrade .csv file by Class (Jupiter doesn't allow import for multiple classes)
  2. Enter the required information in the fields below.
    Find the course name and class tag in Jupiter under SetUp > Classes/Subjects
    For example, Course: AP Physics Tag: PhysicsE 1/2
  3. Download processed .csv and Import processed file into Jupiter
